----------------
 
**地址：**  0x4000a400
 
.. figure:: ../../picture/pwm_pwm_int_config.svg

    | 位       | 名称                         |权限    | 复位值      | 描述                         |

    | 31:14    | RSVD                         |        |             |                              |
    +----------+------------------------------+--------+-------------+------------------------------+

    | 7:6      | RSVD                         |        |             |                              |
    +----------+------------------------------+--------+-------------+------------------------------+
    | 5:0      | pwm_interrupt_sts            | r      | 6'd0        | PWM channel interrupt status |
